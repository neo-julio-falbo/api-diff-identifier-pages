API Polling Micro-SaaS

About the Project

This project provides a scalable, reliable, and efficient Micro-SaaS platform for transforming traditional API polling into an event-driven workflow. Designed for APIs without webhook support, it enables seamless integration by continuously monitoring APIs, detecting changes, and delivering real-time webhook notifications.

Key Features

  • Dynamic API polling intervals
  • Change detection using CRC32 and JSON diffing
  • Configurable rate limits
  • Real-time webhook delivery
  • Modular and scalable architecture using Spring Boot and Quartz
